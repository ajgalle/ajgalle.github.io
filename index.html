9/06/2017 Just a quick update to showcase some WebForms projects I've been working on. First up is a ADO.net/C#/ASP.net app I made to help warehouse workers log what comes in and out. The tricky parts were with how to get all the techologies to play nice together. For example, getting an external javascript file to use jQuery to grab a server side control to animate it was oddly difficult. Next up I'll try a MVC project with Entity framework.
7/14/2017 Recently I've had a choice on how to go forward. Either I could move on to starting Angular 2, or try to learn C#. I keep running into frustrating problems by not knowing a backend server side language and it seems there a lot of jobs in my area that want C#, so I've been moving in that direction. The material is pretty dense, so it will take a while before I have anything worthwhile that I can upload, but hopefully soon. Right now I've moved from console apps to simple drag and drop forms, but I really want to get into the web development using ASP.net with C#, and it seems that path leads to MVC 5 and SQL, which is really complicated for me right now. Slow and steady.
5/26/2017 Finally uploaded some of my APIs utalizing projects from freeCodeCamp. In the meantime, please enjoy my RevaturePro project, Foodar. It incorporates Javascript to check zipcodes, calculate totals, and more. It also uses some new tricks they taught me with fetching form data and writing text over images. If you haven't heard of them, you should check them out.
5/04/2017 Finally I am ready to start making new projects with freeCodeCamp that use APIs, JavaScript, and more! They estimate it will take over 100 hours to complete these tasks. I redesigned the site today to prepare to showcase them. Now the site uses Flexbox and some CSS3 animations.
5/02/2017 Today I finally finished the last of the "think like a programmer" tasks on freeCodeCamp. That's things like using for loops to iterate through arrays, reversing strings, getting my feet wet with regular expressions and other basic algorithim concepts. I'm very much looking forward to building something so I can apply all this.
